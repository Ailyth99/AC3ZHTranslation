# 自定义寒蝉全圆体 (Custom ChillRound Font)

这是一个基于“寒蝉全圆体”修改的字体文件：`custom_ChillRound.ttf`。

主要修改内容是调整了字体的宽度，以适应AC3特定显示需求。
